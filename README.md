# vimeo-to-bunny
Copy videos from Vimeo to BunnyCDN, automatically.

The script is python based.

## Usage
Open `bridge.py` in your code editor and replace following variables.

* `VIMEO_API_KEY`
* `BUNNY_STREAM_RW_API`

Then run the script from commandline.

`python3 bridge.py`

## Requirements

* Python 3
* Python `Requests` Package
* Python `OS` Package
